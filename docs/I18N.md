# ClipForge i18n Contract

ClipForge ships user-facing copy in `zh-CN` and `en-US`.

## Dictionary

- Runtime dictionaries live in `src/i18n/locales/`.
- Every key added to one locale must be added to every locale.
- Keep technical identifiers, error codes, log fields, and user content out of dictionaries.

## Checks

Run before release:

```bash
pnpm check:i18n
```

This command verifies locale key alignment, validates referenced keys, and reports hardcoded user-copy candidates.

Use strict hardcoded-copy mode during full migration:

```bash
CLIPFORGE_I18N_STRICT=1 pnpm check:i18n
```
