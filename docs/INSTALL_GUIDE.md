# ClipForge 内测安装指南

## 下载与安装

### 1. 下载安装包

下载 `ClipForge_0.1.0_aarch64.dmg` 文件。

### 2. 打开 DMG

双击 DMG 文件，将 `ClipForge.app` 拖拽到 Applications 文件夹。

---

## 绕过 macOS 安全警告

由于 ClipForge 当前处于内测阶段，尚未完成 Apple Developer 签名公证，首次打开时会弹出安全警告。

### 方法一：右键打开（推荐）

1. 在 Finder 中找到 `Applications` 文件夹里的 `ClipForge.app`
2. **右键点击**（或按住 Control 键点击）
3. 在弹出菜单中选择 **"打开"**
4. 系统会弹出确认对话框，点击 **"打开"** 即可

> macOS 会记录你的选择，之后就可以正常双击打开了。

### 方法二：系统设置授权

如果已经双击打开并看到 "无法验证开发者" 的警告：

1. 打开 **系统设置** → **隐私与安全**
2. 在底部找到 "ClipForge.app 已被阻止..."
3. 点击 **"仍要打开"** 按钮
4. 在弹出对话框中确认打开

### 方法三：终端授权（备选）

如果上述方法无效，可以在终端执行：

```bash
xattr -cr /Applications/ClipForge.app
```

这会清除应用的隔离属性，之后即可正常打开。

## 授权辅助功能权限

ClipForge 需要辅助功能权限来实现核心功能：

- 悬浮面板贴近输入控件位置
- 全局快捷键唤起
- 自动粘贴到当前应用

### 授权步骤

1. 打开 ClipForge 后，会弹出权限提示
2. 点击 **"打开系统设置"**
3. 在 **隐私与安全** → **辅助功能** 中
4. 找到 `ClipForge.app`，勾选启用
5. 可能需要重启 ClipForge 使权限生效

如果首次没有弹出提示，可以手动添加：

1. **系统设置** → **隐私与安全** → **辅助功能**
2. 点击左下角 **+** 按钮
3. 选择 `/Applications/ClipForge.app`

## 基本使用

### 全局快捷键

默认快捷键：**Command + Shift + V**

按下快捷键会唤起 ClipForge 悬浮面板，靠近当前输入框位置。

### 系统托盘

ClipForge 会在系统托盘（右上角菜单栏）显示图标：

- 点击托盘图标可打开主面板
- 右键托盘图标有快捷菜单

### 核心功能

- **历史记录**：自动记录剪贴板内容
- **快速粘贴**：选中内容后按 Enter 或点击复制
- **收藏**：标记常用内容
- **内容筛选**：按类型（链接、代码、文本等）快速筛选
- **多屏幕支持**：面板会跟随输入框出现在正确的屏幕

## 常见问题

### Q: 面板没有出现在输入框附近？

检查是否已授权辅助功能权限。如果未授权，ClipForge 无法读取输入控件位置。

### Q: 快捷键不起作用？

1. 检查是否与其他软件快捷键冲突
2. 确认 ClipForge 正在运行（查看托盘图标）
3. 尝试在 ClipForge 设置中重新绑定快捷键

### Q: 面板显示后立即消失？

可能是焦点切换问题。尝试：
- 按住快捷键稍长一点再松开
- 从托盘图标点击打开

### Q: 内容没有自动记录？

确认 ClipForge 正在运行，且剪贴板监听已启用（默认开启）。
