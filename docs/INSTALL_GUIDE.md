# ClipForge 内测安装指南

## 下载与安装

### 1. 下载安装包

下载 `ClipForge_0.1.0_aarch64.dmg` 文件。

### 2. 打开 DMG

双击 DMG 文件，将 `ClipForge.app` 拖拽到 Applications 文件夹。

## 首次打开时的安全提示

ClipForge 当前分发的是 **内测版本**。如果你在 macOS 上看到以下提示：

- `无法验证开发者`
- `ClipForge 已损坏，无法打开。你应该将它移到废纸篓`
- `Apple 无法检查其是否包含恶意软件`

这通常 **不是应用真的损坏**，而是 macOS 的安全机制 `Gatekeeper` 在拦截一个尚未完成 Apple 官方签名/公证的应用。

macOS 会给从浏览器下载的应用附加隔离标记 `com.apple.quarantine`。对于内测版、开源版或未签名构建，这会直接导致首次运行被阻止。

### 方法一：右键打开（首选）

这是最安全、最推荐的处理方式。

1. 在 Finder 中打开 `Applications` 文件夹，找到 `ClipForge.app`
2. 不要双击，改为 **按住 Control 键点击**，或直接 **右键点击**
3. 选择 **“打开”**
4. 在系统弹窗里点击 **“打开”** 或 **“仍要打开”**

完成一次后，macOS 会记住这次授权，之后就可以正常双击启动。

### 方法二：系统设置里允许打开

如果你已经双击过一次，并且看到了“无法验证开发者”或“已被阻止”的提示：

1. 打开 **系统设置** → **隐私与安全**
2. 在页面底部找到与 `ClipForge.app` 相关的阻止提示
3. 点击 **“仍要打开”**
4. 按提示再次确认

### 方法三：终端移除隔离属性（备选）

如果前两种方法都不方便，可以在终端执行：

```bash
xattr -r -d com.apple.quarantine /Applications/ClipForge.app
```

如果应用安装在别的位置，请把命令里的路径替换成实际路径。

这条命令只会移除 macOS 附加的隔离标记，不会修改应用内容本身。

### 不推荐的方法

不要使用下面这类命令去全局关闭 macOS 的安全保护：

```bash
sudo spctl --master-disable
```

这会降低整台机器的安全性，不适合作为内测安装方案。优先使用 `右键打开` 或 `xattr` 清理隔离属性即可。

## 授权辅助功能权限

ClipForge 需要辅助功能权限来实现核心功能：

- 悬浮面板贴近输入控件位置
- 全局快捷键唤起
- 自动粘贴到当前应用

### 授权步骤

1. 打开 ClipForge 后，会弹出权限提示
2. 点击 **"打开系统设置"**
3. 在 **隐私与安全** → **辅助功能** 中
4. 找到 `ClipForge.app`，勾选启用
5. 可能需要重启 ClipForge 使权限生效

如果首次没有弹出提示，可以手动添加：

1. **系统设置** → **隐私与安全** → **辅助功能**
2. 点击左下角 **+** 按钮
3. 选择 `/Applications/ClipForge.app`

## 基本使用

### 全局快捷键

默认快捷键：**Control + V**

按下快捷键会唤起 ClipForge 悬浮面板，靠近当前输入框位置。

你可以在设置中修改这个快捷键；默认文档、安装说明和引导动画统一以 `Control + V` 作为默认唤起入口。

### 系统托盘

ClipForge 会在系统托盘（右上角菜单栏）显示图标：

- 点击托盘图标可打开主面板
- 右键托盘图标有快捷菜单

### 核心功能

- **历史记录**：自动记录剪贴板内容
- **快速粘贴**：选中内容后按 Enter 或点击复制
- **收藏**：标记常用内容
- **内容筛选**：按类型（链接、代码、文本等）快速筛选
- **多屏幕支持**：面板会跟随输入框出现在正确的屏幕

## 常见问题

### Q: 面板没有出现在输入框附近？

检查是否已授权辅助功能权限。如果未授权，ClipForge 无法读取输入控件位置。

### Q: 为什么会提示“已损坏”？

这通常不是程序文件真的损坏，而是 macOS 把未公证的内测应用标记为隔离状态。优先按上面的 `右键打开` 方式处理；如果仍不行，再执行：

```bash
xattr -r -d com.apple.quarantine /Applications/ClipForge.app
```

### Q: 快捷键不起作用？

1. 检查是否与其他软件快捷键冲突
2. 确认 ClipForge 正在运行（查看托盘图标）
3. 尝试在 ClipForge 设置中重新绑定快捷键

### Q: 面板显示后立即消失？

可能是焦点切换问题。尝试：
- 按住快捷键稍长一点再松开
- 从托盘图标点击打开

### Q: 内容没有自动记录？

确认 ClipForge 正在运行，且剪贴板监听已启用（默认开启）。
