# Changelog

All notable changes to ClipForge will be documented here.

## 0.1.0 - 2026-07-03

- Initial Tauri v2 + React desktop application.
- Added local clipboard capture, search, copy-back, favorites, soft delete, and cleanup.
- Added SQLite persistence and cursor-based query flow.
- Added compact quick panel, settings window, status bar menu, and macOS `Control+V` shortcut.
- Added service contracts for future sync, import/export, and MCP integration.
