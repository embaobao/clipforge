# 提案：UI 设计风格升级

## 背景

当前 ClipForge UI 缺少统一的视觉风格：
- 使用自定义 CSS，缺少设计 token 体系
- 风格偏普通，缺少品牌识别度
- 与 pi.dev 等现代工具风格差距明显

参考项目调研结论：
- **pi.dev** 风格：黑白对比、极简、高信息密度
- **EcoPaste** 使用 Ant Design 6 + UnoCSS，组件库成熟
- **TieZ** 提供 5 款主题风格：毛玻璃、笔记本、便利贴、3D 等
- **AGENTS.md** 要求：跨平台中性工具风格，克制、轻量、清晰层级

## 目标

- 建立统一的设计 token 体系（颜色、字体、间距、圆角）
- 参考 **pi.dev** 风格：黑白对比、极简、高信息密度
- 参考 **shadcn/ui** 风格：语义 token、无依赖复制
- 优化快捷面板视觉：紧凑、透明、圆角、毛玻璃效果
- 优化管理窗口视觉：清晰层级、信息密度高

## 非目标

- 不引入重型 UI 库（如 Ant Design、Material UI）
- 不实现多主题切换（后续提案考虑）
- 不实现复杂动画（保持性能）

## 用户价值

- 面板视觉更现代、更原生
- 高信息密度，一屏显示更多内容
- 统一视觉风格，品牌识别度高
- 跨平台一致性体验

## 技术调研结论

### pi.dev 风格特征

- **黑白对比**：主色调黑白，强调对比度
- **极简布局**：去掉冗余装饰，保留核心功能
- **高信息密度**：紧凑间距，一屏显示更多
- **无阴影**：用色块和边框分隔，不用阴影
- **小圆角**：4px~8px 圆角，不夸张
- **等宽字体**：代码和 ID 使用等宽字体

### shadcn/ui Token 体系

```css
/* Semantic Tokens */
:root {
  --background: 0 0% 100%;
  --foreground: 240 10% 3.9%;
  --card: 0 0% 100%;
  --card-foreground: 240 10% 3.9%;
  --primary: 240 5.9% 10%;
  --primary-foreground: 0 0% 98%;
  --secondary: 240 4.8% 95.9%;
  --secondary-foreground: 240 5.9% 10%;
  --muted: 240 4.8% 95.9%;
  --muted-foreground: 240 3.8% 46.1%;
  --accent: 240 4.8% 95.9%;
  --accent-foreground: 240 5.9% 10%;
  --destructive: 0 84.2% 60.2%;
  --border: 240 5.9% 90%;
  --input: 240 5.9% 90%;
  --ring: 240 5.9% 10%;
  --radius: 0.5rem;
}
```

### TieZ 主题风格

1. **极简毛玻璃**：透明背景 + backdrop-filter
2. **笔记本风格**：纸张纹理 + 笔迹感
3. **便利贴风格**：彩色便利贴 + 手写感
4. **3D 动感**：阴影 + 悬浮感
5. **暗黑模式**：深色背景 + 高对比

### ClipForge 选择

**默认风格**：pi.dev + shadcn/ui 混合
- 黑白对比为主
- 语义 token 体系
- 紧凑间距
- 小圆角（6px）
- 无阴影（用边框分隔）
